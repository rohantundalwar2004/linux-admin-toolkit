# Linux Admin Toolkit

A collection of Linux administration scripts written in Bash for DevOps learning.

## Features

- System Health Report
- CPU Monitoring
- Memory Monitoring
- Disk Usage Monitor
- Network Information
- Running Process Monitor
- Backup Manager
- Directory Report
- Uptime Checker

## Technologies

- Bash
- Linux Commands
- Shell Scripting
- Git
- GitHub

## Project Structure

## How to Run

```bash
chmod +x scripts/*.sh
cd scripts
./toolkit.sh
```
